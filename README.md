# imgc
A script for compressing images

# Useages
compress.py destdir [rate]

* destdir : include all image dir
* rate    : compress rate

# Example

```
a :  b : 1.jpg 
         2.jpg  
     c : 1.png 
         2.png 

after runing 

new a :  b : 1.jpg
             2.jpg
         c : 1.png
             2.png
```
